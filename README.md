# Weather-App

🌦️ Weather App

A responsive and dynamic Weather Forecast Web Application built using React, HTML, CSS, and JavaScript, integrated with a real-time weather API to provide accurate weather data for any location worldwide.

🚀 Features

🔍 Search weather by city name

🌡️ Real-time temperature, humidity, and wind speed

☁️ Dynamic weather conditions (Clear, Clouds, Rain, etc.)

⚡ Fast and responsive UI

❌ Error handling for invalid city inputs

🔄 Live data fetched using API integration

🛠️ Tech Stack

Frontend: React, HTML5, CSS3, JavaScript (ES6+)

API: Weather API (e.g., OpenWeatherMap)

Tools: Git, GitHub
